# HAUT_Community

#### 介绍

软件工程项目实践 -- **Gathering Circle** 社交平台--前端后台代码

实现了一个校园社区交流平台，前端后台使用Vue 3.0 + TypeScript等开发，使用了Pure-Admin框架

#### 技术栈

使用到了Vue 3.0 + TypeScript + Element Plus + Vite 等
